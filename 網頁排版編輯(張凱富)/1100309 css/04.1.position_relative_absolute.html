相對定位 position: relative; (進入定位層，會遮蓋正常流向的元素)

1.使用位移屬性時，會以自己原來的位置位移，並且保留自身原來的位置(不會被正常流向的元素佔據)
2.沒有設定任何位移屬性，會留在原地不動
3.位移屬性可以負數

絕對定位 position: absolute; (進入定位層)

1.沒有任何位移屬性，該元素會脫離正常流向獨立出來，後方元素依據 CSS 流動的特性往上流動，在絕對定位的下方角為定位點
2.若祖先元素都找不到定位點 (absolute relative fixed)，最後會以初始容器根元素(html)來定位
3. inline 或 inline-block 元素會被自動轉型為 block
