rem 與 em 的差異

◆ rem

rem = root(html) em = 以 root(html) 網頁原始字形大小 16px 去做縮放

這裡的筆記還沒打完喔 ??
font 1rem=1*16px=16px
font 2rem=2*16px=32px
font 3rem=3*16px=48px

◆ em

em 為父層 font-size 的倍數，父層若沒有設定，則會一直往上尋找至 root(html) 預設的font-size

font 1em=1*16px=16px
font 2em=2*16px=32px
font 3em=3*32px=96px
