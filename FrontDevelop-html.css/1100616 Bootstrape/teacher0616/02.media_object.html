媒體物件 (Media object)

媒體物件 .media 是一個 flex，圖片、.media-body 是彈性項目，.media-body flex:1 分配了剩餘的空間

巢狀

媒體物件可以無限做巢狀，但是我們建議你別作過頭。將巢狀的媒體.media放在父媒體對象的.media-body內。

媒體列表
