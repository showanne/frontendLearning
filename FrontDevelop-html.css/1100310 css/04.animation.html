Transition 轉場

  1. 1.兩組樣式之間漸進式的轉換過程，一個是起始樣式，一個是結束樣式，瀏覽器會將兩個樣式形成補間動畫(Tween)
  2. 2.必須要有狀態觸發動作(狀態有很多種)，最常見 :hover ，程式修改 class id，或是 media query 媒體查詢尺寸的改變
  3. 3.無法載入時自動發生，而且他是一次性，沒有循環效果，必須觸發完再重新觸發一次
