下拉選單 (Dropdowns)

  • data-toggle="dropdown" 觸發 .dropdown-toggle 產生下拉的效果
  • 使用 data-reference="parent" 找到父元素，即可使用 data-offset="x,y" 位移對齊，數值可為負
  • .dropup .dropright .dropleft 設定下拉選單的方向
  • 分類標題 (沒有下拉功能) .dropdown-item-text
    分隔線 ↑ .dropdown-divider
    目前位置 .active
    禁用 .disabled
  • 分離按鈕(Toggle Dropdown) 用 btn-group 包起來
  • 設定 data-display="static" 才能使用響應式對齊
    加上斷點或是更大的下拉選單對齊，請加上 .dropdown-menu{-sm|-md|-lg|-xl}-right / left.
