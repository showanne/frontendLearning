1. 變數（Variables）

$ 符號宣告變數名稱的位置，是它的作用域，就像 JS 的 let 宣告，寫在大括號裡面屬於大括號內的變數，寫在最外面為全域變數。

變數的底線(_) 和 連接線(-) 可以互換使用，$bg_black、$bg-black 兩者是一樣的。

變數可以維持網站的一致姓，可以節省修改成本。

3. 嵌套巢狀（nested）

勿過度使用巢狀，考慮用最少的結構選擇到目標，一般建議三層，不超過四層

過度使用巢狀，編譯後選擇器會有過長的問題，樣式不易閱讀，瀏覽器渲染效果會變差，且 CSS 檔案容量較大。

4. Extend(繼承) % 符號

繼承可以用 CSS 選擇器 或 % 佔位選擇器來命名。

% 佔位選擇器屬於 SASS 的範疇，不會被編譯到 CSS。

相同繼承樣式編譯後會被用群組選擇器聚集在一起。

@extend 沒有參數可以使用。

6. Function（函式）

@function 和 @mixin 很像，差異在於 @function 必須使用 @return 回傳結果（符合CSS規範的計算值），它無法在呼叫時產生一段樣式碼， 而 @mixin 則可以在呼叫時產生一段樣式碼。
