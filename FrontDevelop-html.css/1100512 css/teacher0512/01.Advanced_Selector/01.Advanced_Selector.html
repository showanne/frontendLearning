4. 虛擬類別（偽類）選擇器，:(冒號)

偽類與結構或狀態有關，在選擇結構、狀態上可以賦予樣式

:enabled、:disabled、:focus

:enabled 可啟用操作點選或輸入的狀態

:disabled 停用不可操作狀態

:focus 當表單輸入欄位或元素有 tabindex 屬性成為焦點時套用樣式，成為焦點是指滑鼠點選或按tab

:checked

表單勾選 checkbox 或 radio 狀態時套用樣式。


:target

錨點選擇器，點擊連結會再連結後面產'#idName'，可使用 :target 套用樣式到擁有 id 名稱的元素上
