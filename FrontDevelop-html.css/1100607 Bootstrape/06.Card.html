主要資訊放在 .card-body 裡面可放 .card-title .card-subtitle .card-text .card-link

除了 .card-body 還有 .card-head .card-footer

使用 .row .col 嚴禁在 .col 上加 margin，若要加 margin ，則在 .col 內再包一層 div ，盡量保持網格系統的乾淨

為使不同高度之 .card 看起來一致，先將歌度設 h-100
將 .card-body (原為彈性項目，.card 是彈性盒) 轉為彈性盒 (d-flex)
並將裡面內容(Ex. card-title / card-text / btn) 轉為彈性項目好分配剩餘空間 並將 .card-text 內的內容 以下列三種方式將剩餘的空間進行比例分配給彈性項目：
  • mb-auto → 把空間推出去
  • flex-fill → 把剩餘空間分給自己
  • flex-grow-1 → 把剩餘空間分給自己
Tip：可先在 .col-sm-4 加背景顏色測試高度

圖像覆蓋 .card-img-overlay

將圖像轉換為卡片背景，並疊加卡片的文字。 根據圖像，你可以選擇是否需要額外的樣式或通用類別。

法 1 將四邊位移設定為0就是填滿整個父元素 

  .card-img-overlay {
  position: absolute;
  top: 0;
  right: 0;
  bottom: 0;
  left: 0;  
  padding: 1.25rem;
  }
法 2 將寬高設定為100 也是填滿整個父元素
缺點：無法位移 

  .card-img-overlay {
  position: absolute;
  width: 100%;
  height: 100%;
  padding: 1.25rem;
  }
.card-img-overlay + img + style="object-fit:cover;"


<img src="https://picsum.photos/1920/1080/?random=1"
 class="card-img w-100 h-100" style="object-fit:cover;" >
< div class="card-img-overlay" >
