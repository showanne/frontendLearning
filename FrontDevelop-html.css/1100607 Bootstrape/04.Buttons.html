按鈕 (Buttons)

.btn 是設計為 <button> 元素一起使用。 但是，您也可以在 <a> 或 <input> 元素上使用，若要在 <button> 使用，建議加上 role="button" 適當地傳達他們的目的到螢幕閱讀器等輔助性技術。

a:Link

較大 (btn-lg) 或較小 (btn-sm) 的按鈕

.active (啟用狀態) 與 .disabled (停用狀態)

啟用狀態 (.active（並包括 aria-pressed="true" 屬性）)

為按鈕增加 .active 切換啟用狀態
切換前 link NO active 切換後 Link active
disabled 屬性可以使用在 <button> 上

<a> 沒有 disabled 屬性，但可增加 .disabled
a:Link .disabled

按鈕插件

觸發 data-toggle="button" 功能切換按鈕 active 狀態。

aria-pressed="false" false為未點狀態。

核取方塊和選項按鈕

將 data-toggle="buttons" 加到含有 .btn-group .btn-group-toggle 來切換 <input> 的 checkbox 或 radio 按鈕
