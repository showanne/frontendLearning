Flexbox

Flexbox 設定分為兩部分：彈性盒(flex container)、彈性項目(flex items)

彈性盒設定

1.display:flex

宣告父元素 display:flex 為彈性盒，子元素全部會變成彈性項目(無論什麼類型的盒子)。

彈性盒(父)、彈性項目(子) 各自擁有屬性設定

彈性盒預設屬性裡，不管多少彈性項目都不會換行，但是太多彈性項目由於沒有空間分配，還是會造成爆版

2.彈性盒 flex-direction 的軸向，決定主軸是水平還是垂直

父元素宣告為彈性盒時，產生主軸與次軸對其方式，主軸並非一定是水平流向，次軸也並非一定是垂直流向，要以彈性盒的 flex-direction 決定誰才是主軸。

當 flex-direction: row; 主軸水平流向排列對齊

當 flex-direction: column; 主軸垂直流向排列對齊

2.flex-grow 彈性伸展係數

彈性伸展係數，預設0，是指將剩餘空間分配給彈性項目

設成 1 或 其他數字，會將剩餘的空間進行比例分配給彈性項目，
例如：三欄 flex-grow

flex-grow:0; = flex: 0 1 auto; = flex:initial

flex-grow:1; = flex: 1 1 auto; = flex:auto

3.flex-shrink 彈性收縮係數

彈性收縮係數，預設1， 防止所有彈性項目加起來的總寬度超過彈性和寬度而產生的爆版現象

若 flex-shrink:1 (預設) 不會爆版

若 flex-shrink:0 若彈性項目總寬度大於彈性和寬度時，產生爆版
