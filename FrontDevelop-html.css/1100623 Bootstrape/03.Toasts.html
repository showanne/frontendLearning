吐司 (Toasts)

  1. 使用前須先啟用

    $('.toast').toast(option)
  2. 一開始還未設定觸發可以設 .show 先看看
  3. toast-header 為彈性盒，裡面的 < strong > 設.mr-auto 把 < small > 往最右推
  4. 消失 data-autohide="" 預設為 true (自動消失) ，false (顯示後不會自動消失)
