為使不同高度之 .card 看起來一致，先將歌度設 h-100
將 .card-body (原為彈性項目，.card 是彈性盒) 轉為彈性盒 (d-flex)
並將裡面內容(Ex. card-title / card-text / btn) 轉為彈性項目好分配剩餘空間 並將 .card-text 內的內容 以下列三種方式將剩餘的空間進行比例分配給彈性項目：
  • mb-auto → 把空間推出去
  • flex-fill → 把剩餘空間分給自己
  • flex-grow-1 → 把剩餘空間分給自己
Tip：可先在 .col-sm-4 加背景顏色測試高度
